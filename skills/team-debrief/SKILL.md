# Team Debrief

Update memory with session outcomes before ending.

## When to Use

At the end of a session, or when the user says "wrap up" or "save progress."

## Process

1. Spawn the `session-manager` agent in debrief mode
2. It reviews what was accomplished this session
3. Updates memory files with: decisions made, WIP state, new preferences
4. Flags any retrospective actions from the Meta Agent
