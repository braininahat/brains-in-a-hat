# Team Retrospective

Run a post-task retrospective to improve the team.

## When to Use

After completing a major task, feature, or bug fix batch. The Tech Lead should invoke this automatically.

## Process

1. Spawn the `meta-retro` agent
2. It reviews: what went well, what was missed, agent effectiveness
3. Proposes prompt/checklist updates for underperforming agents
4. Checks for new unowned paths (CODEOWNERS update)
5. Writes retrospective to `.claude/team/retrospectives/`
6. Returns summary to Tech Lead
